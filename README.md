# Multiplication Table Generator

This is a simple Python program that generates the multiplication table of a number entered by the user.

## Features

- Takes an integer input from the user.
- Prints the multiplication table from 1 to 10 for the given number.

## How to Use

1. Clone this repository or download the Python file.
2. Run the Python file using:

bash
python multiplication_table.py
 Enter a number when prompted.

The program will display the multiplication table for that number.

Example

Enter number: 5
5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
5 x 4 = 20
5 x 5 = 25
5 x 6 = 30
5 x 7 = 35
5 x 8 = 40
5 x 9 = 45
5 x 10 = 50
